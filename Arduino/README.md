NGPS_Mods_V0006_quiet_qnethernet.ino is the Arduino sketch code using the qnethernet library to handel ethernet communication.  This "quiet" version has all print statements commented out. 

The modulator PCB uses an Arduino Teensy 4.1 board, but the sketch can be modified to work with a Teensy 3.2.

The sketch allows for turning on/off, changing the frequency and duty cycle of each modulator asynchronously.
